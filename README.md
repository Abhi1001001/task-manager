  <h1>Task Management Application</h1>
  <a href="https://task-manager-assi.netlify.app/">https://task-manager-assi.netlify.app/</a>
  <p>
    This is a Task Management web application built using <strong>React</strong> with
    <strong>Vite</strong> and <strong>JavaScript</strong>.
    The application allows users to manage tasks efficiently with a modern,
    responsive UI styled using <strong>Tailwind CSS</strong>.
    A <strong>Dummy JSON API</strong> is used to simulate backend operations.
  </p>

  <section>
    <h2>Features</h2>
    <ul>
      <li>Add new tasks</li>
      <li>Update task title</li>
      <li>Mark tasks as completed</li>
      <li>Delete tasks</li>
      <li>Responsive and clean UI</li>
      <li>API-based CRUD operations</li>
    </ul>
  </section>

  <section>
    <h2>Tech Stack</h2>
    <ul>
      <li>React (Vite)</li>
      <li>JavaScript (ES6+)</li>
      <li>Tailwind CSS</li>
      <li>Dummy JSON API (Mock Backend)</li>
    </ul>
  </section>

  <section>
    <h2>API Integration</h2>
    <p>
      The application uses a Dummy JSON API to simulate backend functionality such as:
    </p>
    <ul>
      <li>Fetching tasks</li>
      <li>Adding new tasks</li>
      <li>Updating task status</li>
      <li>Deleting tasks</li>
    </ul>
    <p>
      This approach helps demonstrate real-world API handling without setting up a real backend.
    </p>
  </section>

  </section>

  <section>
    <h2>How the App Works</h2>

      <li>User can mark a task as completed</li>
      <li>User can edit task details</li>
