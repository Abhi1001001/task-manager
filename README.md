<h1>Task Management Application</h1>
  <p>
    This is a Task Management web application built using
    <strong>React</strong> with <strong>Vite</strong> and <strong>TypeScript</strong>.
    The app allows users to manage daily tasks efficiently with a clean and responsive UI
    styled using <strong>Tailwind CSS</strong>.
  </p>
  <section>
    <h2>Features</h2>
    <ul>
      <li>Add new tasks</li>
      <li>Update existing tasks</li>
      <li>Mark tasks as completed</li>
      <li>Delete tasks</li>
      <li>Responsive and modern UI</li>
      <li>Backend integration using Dummy JSON API</li>
    </ul>
  </section>
  <section>
    <h2>Tech Stack</h2>
    <ul>
      <li>React (Vite)</li>
      <li>TypeScript</li>
      <li>Tailwind CSS</li>
      <li>Dummy JSON API (Mock Backend)</li>
      <li>JavaScript (ES6+)</li>
    </ul>
  </section>
  <section>
    <h2>Dummy API Usage</h2>
    <p>
      The application uses a Dummy JSON API to simulate backend operations such as:
    </p>
    <ul>
      <li>Fetching task list</li>
      <li>Adding new tasks</li>
      <li>Updating task status</li>
      <li>Deleting tasks</li>
    </ul>
    <p>
      This helps demonstrate real-world API interaction without a real backend.
    </p>
  </section>
  <section>
    <h2>Installation & Setup</h2>
    <pre>
# Clone the repository
git clone https://github.com/your-username/task-management-app.git

# Navigate to project folder

cd task-management-app

# Install dependencies

npm install

# Start development server

npm run dev
</pre>
  </section>

    <h2>How It Works</h2>

      <li>User can update task titles</li>
